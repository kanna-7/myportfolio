# myportfolio

This repository contains a simple static portfolio site added to the workspace root.

Files added at repository root:

- `index.html`
- `css/styles.css`
- `js/script.js`

How to view locally (PowerShell):

```powershell
# open in default browser (from repo root)
Start-Process index.html

# or serve the repo root with a static server
npx http-server -c-1
```

Customization:
- Replace "Your Name" and edit content in `index.html`.
- Tweak colors and layout in `css/styles.css`.
- Hook the contact `form` in `js/script.js` to your backend or an email service.
